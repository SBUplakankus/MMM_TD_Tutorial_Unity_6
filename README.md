# Tower Defense Tutorial - Massive Melt Media

---

## About This Tutorial
The goal of this tutorial is the help beginner developers take the next step in their game development journey
by teaching them some key concepts and design patterns that can be used in any future games.

The main focus is on more high level game architecture and design choices as oppossed to specfic tower defence game mechanics.
You will learn the benefits of a clean project structure and pick up some cool tips tricks Unity developers do to improve the
performance and scalability on larger projects. 

---

## What You'll Learn
- Decoupling Systems through use of the Observer Pattern
- Efficient instantiation and destruction of objects through Object Pooling
- A Data Driven design approach through Scriptable Objects and Databases
- Improving game performance through a custom Update Manager
- Removing magic variable dependancy through a Game Constants class

---

## Prerequisites
- Roughly a year of experience in the Unity engine *(You've done the beginner stuff)*
- Familiarity with C# or a similar language
- Developed using Unity 6.3 using the 3D URP but should work on similar or later versions
