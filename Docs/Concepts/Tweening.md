# Tweening with PrimeTween

## Purpose
Tweens are what make a game feel responsive and look polished.
A Tween smoothly interpolates a value over time, such as position, scale, rotation, or opacity.

Effects such as a button growing on mouse enter, text growing, UI elements fading in and out etc. are all driven by Tweens.
PrimeTween is a free and lightweight tweening package for Unity, and the one we will use in this course.

More specific Tweening tutorials can be found on my YouTube channel, this will just be a simple introduction for this course.

---

## Implementation

We will be using Tweens to animate the health bars above our enemies and to animate the UI elements in our game.
Tweens will be used only for visual feedback and will not affect gameplay logic.

- Enemy Healthbars will shake and slide down when damage is taken.
- UI Buttons will grow when entered and shrink on mouse pointer exit.
